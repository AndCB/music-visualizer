## Try it out

you can try this app here: https://andcb.github.io/music-visualizer/

## Description

This is a very simple music visualization app. It uses the web audio API to generate the visualizations of the songs you add to the playlist on the top left.


This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
